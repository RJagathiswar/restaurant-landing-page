# 🍴 Delish Restaurant Landing Page

A responsive restaurant landing page built using **HTML, CSS, JavaScript, and Bootstrap**.

## 🚀 Features
- Responsive design
- Sticky navbar
- Hero banner
- About section
- Menu highlights
- Gallery
- Contact form
- Footer



## ⚡ Run
1. Extract folder.
2. Open `index.html` in browser.
3. Or open in VS Code with Live Server.
